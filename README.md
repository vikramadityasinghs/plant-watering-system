# Plant watering system


## Overview
To create a self-watering plant system using a soil sensor, water pump and nodemcu 8266 and relay module.

## Prerequisites
* C++
* Arduino IDE
* Adafruit.io
* ESP8266WiFi library
* Adafruit_MQTT_Client library
* basic knowledge about nodemcu 8266.
