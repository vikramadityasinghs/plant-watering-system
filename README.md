# Automatic plant watering system
<hr>
## Overview
To create a self-watering plant system using a soil sensor, water pump and nodemcu 8266 and a relay module.
<hr>
## Prerequisites
<ol>
<li>C++</li>
<li>Arduino IDE</li>
<li>Adafruit.io</li>
<li>ESP8266WiFi library</li>
<li>Adafruit_MQTT_Client library</li>
  </ol>
